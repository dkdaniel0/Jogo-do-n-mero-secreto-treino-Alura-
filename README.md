<h1>Jogo do Número Secreto</h1>

<h2> Sobre</h2>
<p>Este projeto é um jogo interativo onde o objetivo é descobrir um número secreto gerado aleatoriamente. O jogador recebe dicas se o número secreto é maior ou menor do que o palpite, até acertar.</p>

<h2> Como Jogar</h2>
<p>1. Abra o arquivo HTML em seu navegador.</p>
<p>2. Digite um número no campo indicado.</p>
<p>3. Clique no botão para verificar seu palpite.</p>
<p>4. Siga as dicas para descobrir se o número secreto é maior ou menor.</p>
<p>5. Continue jogando até acertar o número!</p>
